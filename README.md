# Export-Fidelity-Stock-Lots
Exports stock lots from the positions page of a fidelity account as a CSV.

1. Navigate to the positions page for an account on Fidelity
2. Open the browser console (usually Shift + Ctrl + J)
3. Paste the script into the browser console
4. Type expandRows() and hit enter
5. Let the page load
6. Type main() and hit enter

Note: I am by no means a JavaScript dev and this is probably a rough implementation at best.
